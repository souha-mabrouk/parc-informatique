# Application de Gestion de Parc Informatique

Projet de stage : développement d'une application de bureau pour la gestion d'un parc informatique et réseau d'entreprise.

# Description

Cette application permet de gérer efficacement les équipements informatiques et réseaux d'une entreprise : ordinateurs, imprimantes, caméras, routeurs, NVR, etc.  
Elle est développée en C++ avec Qt pour l'interface graphique, et utilise une base de données SQL pour stocker les informations.

# Fonctionnalités principales

-  **Gestion des équipements** : ajout, modification, suppression.
-  **Catégorisation** : organisation des équipements par catégorie (réseau, vidéosurveillance, serveurs...).
-  **Départements** : gestion des départements et affectation des équipements.
-  **Maintenance** : suivi des opérations de maintenance.
-  **Requêtes et réclamations** : interface pour signaler des pannes ou anomalies.
-  **Gestion des utilisateurs** : rôles, droits d'accès (admin, utilisateur...).
-  **Recherche et filtres** : trouver rapidement un équipement ou un utilisateur.

# Technologies utilisées

- **Langage** : C++
- **Framework GUI** : Qt 5/6
- **Base de données** : SQL
- **Système** : Windows
  
 # Installation & Exécution

 1. Cloner le projet :
 git clone https://github.com/souha-mabrouk/parc-informatique.git
 2. Ouvrir le fichier .pro avec Qt Creator.
 3. Configurer la connexion à la base de données.
 4. Compiler et exécuter.
